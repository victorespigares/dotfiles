= Dotfiles

== Installation

    git clone https://github.com/victorespigares/dotfiles.git ~/dotfiles
    cd ~/dotfiles
    ./makesymlinks.sh
    ~git submodule update --init~

	Don't forget to add Zsh, [oh-my-zsh](https://github.com/robbyrussell/oh-my-zsh) and [Zsh-z](https://github.com/agkozak/zsh-z) for more awesomeness.
